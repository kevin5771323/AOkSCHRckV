# 旅游信息管理系统 java1

## 项目概述

旅游信息管理系统是基于Spring Boot和Vue的前后端分离架构设计。该系统旨在为用户提供一站式的旅游信息服务，包括旅游新闻、景区信息、美食信息、旅游路线以及在线留言等。

## 技术栈

- 前端：Vue
- 后端：Java、Spring Boot
- 数据库：MySQL
- 中间件：MyBatis

## 功能模块

### 前台功能

- 展示旅游新闻
- 展示景区信息
- 展示美食信息
- 展示旅游线路
- 在线留言

### 后台管理员功能

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

### 普通用户功能

- 浏览旅游线路
- 管理订单信息

## 系统角色

- 管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 8+、Spring Boot 2.x、MySQL 5.x

## 部署步骤

1. 准备好运行环境，包括Java环境和MySQL数据库。
2. 导入数据库SQL文件以初始化数据。
3. 部署后端服务，确保Spring Boot服务可以正常运行。
4. 部署前端项目，确保Vue应用可以正常访问后端API。
